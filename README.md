# 🚀 TechTogether Platform

## 📌 Overview
TechTogether is a web-based platform designed to connect job-seeking coders with tech companies.  
It enables seamless interaction between candidates and recruiters through job applications, resume uploads, and interview scheduling.

---

## 👨‍💻 Features

### 👩‍💻 Coder Panel
- View available job openings
- Apply for jobs
- Upload resumes
- Track application status

### 🏢 Admin Panel (Companies)
- View uploaded resumes
- Manage job postings
- Review candidates
- Schedule interviews

---

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot
- **Frontend:** HTML, CSS
- **Database:** MySQL
- **Tools:** Spring Tool Suite (STS), Eclipse
